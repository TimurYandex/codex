# Ball impact simulator (Python 3.14 + arcade)

Проект по плану из `plan_py.md`.

## Как запустить (минимальная проверка)
1. Убедитесь, что `python` соответствует версии `3.14`.
2. Убедитесь, что пакет `arcade` установлен в глобальный Python (виртуальное окружение не используем).
3. Запуск:
   - из корня репозитория: `python python_arcade_sim/main.py`

## Текущий статус
Сейчас реализован только каркас окна `arcade` (baseline), без физики/рендера симулятора. Дальше подключаем шаги из `steps.md`.
