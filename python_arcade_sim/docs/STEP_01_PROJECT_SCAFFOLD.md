# STEP_01: Каркас проекта (Python 3.14 + arcade)

Цель шага: создать минимальную “базу” проекта, чтобы:
- код мог запускаться на Python 3.14;
- дальнейшие шаги из `steps.md` могли добавлять физику/рендер/UI, не меняя каркас.

## Принятые договоренности
1. Физическое ядро будет независимым от `arcade` (рендер/UI только читает готовые данные симуляции).
2. Виртуальное окружение не используем: `arcade` устанавливается в глобальный Python.
3. Дополнительные модули, если понадобятся, устанавливаются через человека с привилегиями администратора; список нужно фиксировать в `requirements.txt`.

## Созданная структура
Папки:
- `app/` (окно/представление)
- `config/` (константы)
- `physics/` (модель, поверхность, контакт, мяч)
- `render/` (отрисовка)
- `ui/` (состояние и панели)
- `validation/` (checks/валидация)
- `utils/` (утилиты)
- `docs/` (документация шагов)
- `tests/` (тесты)

Файлы (минимальные для baseline):
- `main.py` — стартовое окно `arcade`
- `app/window.py` — базовый `SimulationWindow` (фон + текст)
- `README.md` — как запустить baseline
- `.gitignore` — игнор кэшей/IDE

## Как проверить
Запустить:
- `python python_arcade_sim/main.py`

Ожидаемо: должно открыться окно `arcade` с текстом baseline.
