# Шаг 1: Каркас проекта

## Структура папок

```
python_arcade_sim/
├── app/          # Главное окно и цикл приложения
├── config/       # Константы и конфигурация
├── docs/         # Документация шагов
├── physics/      # Физическое ядро (независимо от arcade)
├── render/       # Рендеринг
├── tests/        # Тесты
├── ui/           # UI компоненты
├── utils/        # Утилиты
├── validation/   # Валидация параметров
├── .gitignore
├── main.py       # Точка входа
├── README.md
└── requirements.txt
```

## Ключевые договоренности

1. **Физика независима от arcade**: код физики (`physics/`) не импортирует `arcade` и не зависит от рендеринга.
2. **Python 3.14 + arcade**: последняя версия `arcade` в глобальном Python.
3. **Модульность**: структура соответствует смысловым разделам (physics/, render/, ui/, config/, validation/, utils/).
