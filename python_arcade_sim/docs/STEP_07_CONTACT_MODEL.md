# Шаг 7: Физика контакта — нормаль Fn и трение Ft (stick-slip)

## Цель

Реализовать расчёт нормальной и касательной сил контакта мяча с поверхностью, включая stick-slip переход.

## Созданные модули

### `physics/contact.py`

#### Геометрия контакта

Для каждого узла поверхности в пределах радиуса мяча:

1. **Расстояние до центра мяча:**
   ```
   dd = x_ball - x_i
   ```

2. **Y поверхности мяча:**
   ```
   y_ball_surface = y_ball - sqrt(r² - dd²)
   ```

3. **Penetration (проникновение):**
   ```
   δ = u_y[i] - y_ball_surface
   ```
   - δ > 0: узел внутри мяча (контакт активен)
   - δ ≤ 0: нет контакта

#### Нормальная сила Fn

Закон Герца с демпфированием:

```
F_n = k_c * δ^p + c_c * max(0, v_rel_n)
```

где:
- `k_c` — жёсткость контакта, Н/м^p
- `p` = 1.35 — показатель степени (нелинейная упругость)
- `c_c` — демпфирование контакта, Н·с/м
- `v_rel_n` = -(v_ball_y - v_node_y) — относительная нормальная скорость

**Демпфирование действует только при сближении** (v_rel_n > 0), что предотвращает "прилипание" мяча к поверхности.

**Защитный кап:** F_n ≤ K_FORCE_CAP (1e5 Н)

#### Касательная сила Ft (stick-slip)

Модель трения Кулона с накоплением stick-смещения:

1. **Относительная касательная скорость:**
   ```
   v_rel_t = (v_ball_x - ω*r) - v_node_x
   ```

2. **Накопление stick-смещения:**
   ```
   s += v_rel_t * dt
   ```

3. **Пробная сила:**
   ```
   F_t_trial = -k_s * s
   ```

4. **Критерий stick-slip:**
   - Если |F_t_trial| ≤ μ_s * F_n: **stick**
     - F_t = F_t_trial
   - Иначе: **slip**
     - F_t = -sign(v_rel_t) * μ_k * F_n
     - s пересчитывается: s = -F_t / k_s

5. **Метрики:**
   - slip_time — время проскальзывания
   - is_slipping — флаг активного проскальзывания

#### Суммирование сил

```
Fn_total = Σ F_n[i]  (по всем активным узлам)
Ft_total = Σ F_t[i]  (по всем активным узлам)
```

Давление в узле:
```
P[i] = F_n[i] / (dx * depth)
```

#### Защита от численной нестабильности

1. **safe_sqrt** для r² - dd² (защита от отрицательных значений при dd ≈ r)
2. **Кап сил:** |F_n|, |F_t| ≤ K_FORCE_CAP
3. **Сброс stick-смещения** когда нет контакта (s = 0)
4. **Ограничение s:** |s| ≤ 0.01 м

## Структуры данных

### ContactParams

| Параметр | Описание | Значение по умолчанию |
|----------|----------|----------------------|
| k_c | Жёсткость контакта | 1e6 Н/м^p |
| c_c | Демпфирование | 100 Н·с/м |
| k_s | Касательная жёсткость | 1e5 Н/м |
| mu_s | Трение покоя | 1.0 |
| mu_k | Трение скольжения | 0.5 |
| p | Показатель степени | 1.35 |

### ContactInput

Входные данные для расчёта:
- ball_x, ball_y, ball_r — позиция и радиус мяча
- ball_v_x, ball_v_y, ball_omega — скорость и вращение мяча
- surface — состояние поверхности
- eq_params — эквивалентные параметры
- contact_params — параметры контакта
- dt — шаг времени

### ContactResult

Результат расчёта:
- is_active — контакт активен
- fn_total, ft_total — суммарные силы
- max_penetration — максимальное проникновение
- active_nodes — индексы активных узлов
- pressure — давление в узлах
- slip_velocity — скорость проскальзывания
- stick_displacement — накопленное stick-смещение
- is_slipping — проскальзывание активно
- slip_time — время проскальзывания

## Ожидаемые эффекты

1. **При ударе мяча:**
   - Активируются узлы в пределах радиуса мяча
   - Возникает нормальная сила Fn (отталкивание)
   - Возникает касательная сила Ft (трение)

2. **Stick-режим (малая скорость):**
   - Мяч "прилипает" к поверхности
   - Ft пропорциональна stick-смещению
   - Энергия накапливается в упругой деформации

3. **Slip-режим (большая скорость):**
   - Мяч проскальзывает
   - Ft ограничена μ_k * Fn
   - Энергия рассеивается в тепло

4. **Переход stick → slip:**
   - Когда |F_t_trial| > μ_s * F_n
   - Происходит резкое изменение силы
   - Может вызывать колебания (эффект "скрипа")
