# Шаг 12: Рендер — графики по времени

## Цель

Реализовать отрисовку графиков истории симуляции: Fn, Ft, def, slip, ω, vx, vy.

## Созданные модули

### `render/graphs.py`

Модуль с классом `GraphPanel` и функцией `draw_graphs()` для отрисовки графиков.

#### Класс `GraphPanel`

Панель для отрисовки одного графика.

**Методы:**

| Метод | Описание |
|-------|----------|
| `__init__(x, y, width, height, title)` | Инициализация панели |
| `draw_frame()` | Нарисовать рамку и заголовок |
| `draw_grid(x_min, x_max, y_min, y_max)` | Нарисовать сетку |
| `draw_line(points, color, t_min, t_max, v_min, v_max, thickness)` | Нарисовать линию по точкам |

**Преобразование координат:**
- `_time_to_pixel(t, t_min, t_max)`: время → пиксель по X
- `_value_to_pixel(v, v_min, v_max)`: значение → пиксель по Y

#### Функция `draw_graphs()`

Отрисовывает все 7 графиков истории:

| График | Цвет | Диапазон (авто) |
|--------|------|-----------------|
| Fn (N) | Зелёный | 0 .. max(Fn) |
| Ft (N) | Оранжевый | -50 .. 50 |
| Deflection (m) | Фиолетовый | 0 .. max(def) |
| Slip (m/s) | Коричневый | -10 .. 10 |
| Omega (rad/s) | Голубой | -100 .. 100 |
| Vx (m/s) | Синий | -20 .. 20 |
| Vy (m/s) | Красный | -20 .. 20 |

**Автоматическое масштабирование:**
- Диапазоны вычисляются по данным истории
- Добавляется 10% запас по краям
