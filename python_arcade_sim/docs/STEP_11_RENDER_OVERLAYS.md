# Шаг 11: Рендер — оверлеи векторов и контактного пятна

## Цель

Реализовать отрисовку оверлеев: векторы сил (Fn, Ft), вектор скорости, контактное пятно с картой давления.

## Созданные модули

### `render/overlays.py`

Модуль с функциями для отрисовки оверлеев.

#### Функции

| Функция | Описание |
|---------|----------|
| `draw_vector(start_x, start_y, vx, vy, color, scale, thickness)` | Нарисовать вектор (стрелку) |
| `draw_force_vectors(snapshot, renderer, scale)` | Векторы Fn (зелёный) и Ft (оранжевый) |
| `draw_velocity_vector(snapshot, renderer, scale)` | Вектор скорости мяча (синий) |
| `draw_contact_patch(snapshot, renderer)` | Контактное пятно с картой давления |
| `draw_overlays(snapshot, renderer, show_fn, show_ft, show_v, show_contact)` | Нарисовать все оверлеи |

#### Вектор сил

**Нормальная сила Fn:**
- Цвет: зелёный
- Направление: вверх (перпендикулярно поверхности)
- Длина пропорциональна величине силы

**Касательная сила Ft:**
- Цвет: оранжевый
- Направление: горизонтально (по касательной)
- Длина пропорциональна величине силы

#### Вектор скорости

- Цвет: синий
- Направление: по касательной к траектории
- Длина пропорциональна скорости

#### Контактное пятно

- Активные узлы (в контакте) рисуются кружками
- Цвет зависит от давления:
  - Жёлтый: низкое давление
  - Оранжевый: среднее давление
  - Красный: высокое давление
