# Шаг 6: Физика шипов — модель наклона и влияние на трение

## Цель

Реализовать упрощённую модель шипов с динамикой наклона θ и влиянием на трение/контакт.

## Созданные модули

### `physics/spikes.py`

#### Модель шипов

Физическая модель:
```
θ¨ = f_th - k_sh * θ - c_sh * θ_dot
```

где:
- `θ` — наклон шипов, рад
- `θ_dot` — скорость наклона, рад/с
- `f_th` — движущая сила от касательного воздействия
- `k_sh` — жёсткость шипов
- `c_sh` — демпфирование наклона

#### Движущая сила

```
f_th = pg * F_t * scale
```

- `pg` — плотность шипов (1000 шт/м², внутренний параметр)
- `F_t` — касательная сила контакта
- `scale` — коэффициент направления:
  - `out`: 1.5 (шипы торчат наружу — сильнее реагируют)
  - `in`: 0.5 (шипы внутрь — слабее реагируют)

#### Ограничения

| Параметр | Ограничение | Физический смысл |
|----------|-------------|------------------|
| `θ` | `[-θ_max, +θ_max]` | Максимальный наклон (~28°) |
| `θ_dot` | `[-θ_max/dt, +θ_max/dt]` | Защита от чрезмерной скорости |
| `F_t_sh` | `[-K_FORCE_CAP, +K_FORCE_CAP]` | Кап дополнительной силы |

#### Влияние на трение

```
mu_multiplier = 1.0 + k_theta * |θ|  (для out)
mu_multiplier = 1.0 - k_theta * 0.5 * |θ|  (для in)
```

- `k_theta = 2.0` — коэффициент усиления
- `mu_s_new = mu_s_base * mu_multiplier`
- `mu_k_new = mu_k_base * mu_multiplier`
- Ограничение: `mu_k_new <= mu_s_new`

#### Дополнительная касательная сила

```
F_t_sh = k_sh * θ * sign(v_rel_t)
```

- Направлена против движения
- Пропорциональна наклону

## Функции

### `compute_spikes_dynamics(state, input_params)`

Вычислить динамику шипов на один шаг.

**Вход:**
- `state.theta`, `state.theta_dot` — текущее состояние
- `input_params.spike_params` — параметры шипов
- `input_params.mode` — режим (NONE/OUT/IN)
- `input_params.ft_contact` — касательная сила контакта
- `input_params.v_rel_t` — относительная скорость
- `input_params.dt` — шаг времени

**Выход:**
- `SpikesOutput` с обновлёнными `theta`, `theta_dot`, `mu_multiplier`, `ft_additional`

### `init_spikes_state()`

Инициализировать нулевое состояние шипов.

### `apply_spikes_to_friction(mu_s_base, mu_k_base, spikes_state)`

Применить влияние шипов к коэффициентам трения.

## Ожидаемые эффекты

1. **При контакте мяча:**
   - Шипы наклоняются в направлении движения мяча
   - Наклон создаёт дополнительную силу трения
   - Трение увеличивается (для out) или уменьшается (для in)

2. **После отрыва:**
   - Шипы возвращаются в исходное положение (θ → 0)
   - Трение возвращается к базовому значению

3. **Зависимость от режима:**
   - `out`: шипы торчат наружу → сильнее сцепление
   - `in`: шипы внутрь → слабее сцепление
   - `none`: шипы не активны → без влияния
