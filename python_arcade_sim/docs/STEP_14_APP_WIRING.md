# Шаг 14: UI + Window — связать всё воедино

## Цель

Интегрировать все компоненты (физика, рендер, UI) в единое приложение с полным циклом симуляции.

## Обновлённые модули

### `app/window.py` (полная реализация)

Класс `SimulationWindow` интегрирует все компоненты.

#### Компоненты

| Компонент | Описание |
|-----------|----------|
| `model: PhysicsModel` | Физическая модель |
| `renderer: Renderer` | Рендерер симуляции |
| `ui_state: UIState` | Состояние интерфейса |

#### Обработчики событий

| Метод | Описание |
|-------|----------|
| `on_draw()` | Отрисовка кадра (симуляция + UI) |
| `on_update(delta_time)` | Обновление физики |
| `on_mouse_press()` | Обработка кликов (кнопки, пауза) |
| `on_mouse_motion()` | Отслеживание курсора для hover |
| `on_key_press()` | Горячие клавиши |

#### Методы

| Метод | Описание |
|-------|----------|
| `reset_simulation()` | Сброс и инициализация симуляции |
| `_draw_ui_panels()` | Отрисовка панелей управления |
| `_draw_status_text()` | Отрисовка статуса |
| `_is_action_button_click()` | Проверка клика по кнопке |
| `_run_self_test()` | Самопроверка (sanity checks) |

## Управление

### Мышь

| Действие | Описание |
|----------|----------|
| ЛКМ по кнопке | Выполнить действие кнопки |
| ЛКМ по окну | Пауза/продолжение |
| ЛКМ после FINISHED | Перезапуск |

### Клавиатура

| Клавиша | Действие |
|---------|----------|
| Пробел | Пауза/продолжение |
| R | Перезапуск симуляции |
| G | Переключить графики |
| O | Переключить оверлеи |
| Escape | Выход |
| 1 | Пресет classic |
| 2 | Пресет inv |
| 3 | Пресет hard |

### Кнопки действий

| Кнопка | Действие |
|--------|----------|
| Run/Pause | Запуск/пауза симуляции |
| Compare (0/3) | Режим сравнения (до 3 прогонов) |
| Self Test | Самопроверка |
| Zoom | Переключение масштаба |

## Интеграция компонентов

```
SimulationWindow
├── PhysicsModel (физика)
│   ├── reset(params)
│   └── step(dt)
├── Renderer (рендер)
│   ├── render(snapshot, surface_params, show_overlays)
│   └── set_scale(scale)
├── UIState (состояние UI)
│   ├── to_simulation_params()
│   └── cycle_view_scale()
└── UI Panels (отрисовка)
    ├── draw_collision_panel()
    ├── draw_ball_panel()
    ├── draw_surface_panel()
    ├── draw_animation_panel()
    └── draw_action_buttons()
```

## Поток данных

1. **UI → Физика:**
   ```
   UIState.to_simulation_params() → PhysicsModel.reset()
   ```

2. **Физика → Рендер:**
   ```
   PhysicsModel.get_render_snapshot() → Renderer.render()
   ```

3. **Физика → UI:**
   ```
   PhysicsModel.get_metrics() → draw_status_text()
   ```
