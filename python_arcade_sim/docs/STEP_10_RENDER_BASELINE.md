# Шаг 10: Рендер — базовая отрисовка разреза и мяча

## Цель

Реализовать базовый рендеринг симуляции: мяч, поверхность (слои в разрезе), базовая сетка.

## Созданные модули

### `render/renderer.py`

Класс `Renderer` для отрисовки симуляции через arcade.

#### Основные методы

| Метод | Описание |
|-------|----------|
| `__init__(width, height, scale)` | Инициализация рендерера |
| `set_scale(scale)` | Установить масштаб отображения |
| `world_to_screen(x, y)` | Преобразование координат мира (м) в экранные (пиксели) |
| `draw_background()` | Нарисовать фон |
| `draw_grid()` | Нарисовать базовую сетку и оси |
| `draw_surface(surface_params, snapshot)` | Нарисовать поверхность (слои) |
| `draw_ball(snapshot)` | Нарисовать мяч с меткой вращения |
| `render(snapshot, surface_params, metrics)` | Полная отрисовка кадра |

#### Система координат

**Мир (метры):**
- X: 0 в центре (точка удара), положительное вправо
- Y: 0 на уровне недеформированной поверхности, положительное вверх

**Экран (пиксели):**
- X: 0 слева, положительное вправо
- Y: 0 сверху, положительное вниз (arcade)

**Преобразование:**
```python
screen_x = center_x + x * pixels_per_meter
screen_y = center_y + y * pixels_per_meter
```

где `pixels_per_meter = 2000 * scale`

#### Отрисовка мяча

1. **Основной круг:** белый с чёрной обводкой
2. **Метка вращения:** красная точка на "экваторе" мяча, угол зависит от `phi`
3. **Стрелка направления:**
   - `omega > 0` (CCW): стрелка влево
   - `omega < 0` (CW): стрелка вправо

#### Отрисовка поверхности

1. **Слои:** прямоугольники разных цветов
2. **Цвета слоёв:** циклически из палитры (зелёный, синий, жёлтый, розовый, лавандовый)
3. **Деформированная поверхность:** линия по узлам (если есть данные в snapshot)

### `app/window.py` (обновлён)

Класс `SimulationWindow` для управления окном arcade.

#### Обработчики событий

| Метод | Описание |
|-------|----------|
| `on_draw()` | Отрисовка кадра |
| `on_update(delta_time)` | Обновление физики |
| `on_mouse_press()` | Пауза/продолжение по клику |
| `on_key_press()` | Управление клавишами |

#### Управление

| Клавиша | Действие |
|---------|----------|
| ЛКМ | Пауза/продолжение |
| Пробел | Пауза/продолжение |
| R | Перезапуск симуляции |
| Escape | Выход |
