# Шаг 2: Окружение и зависимости (Python 3.14 + arcade)

## Зависимости

- **Python**: 3.14 (глобальная установка, без виртуального окружения)
- **arcade**: последняя версия (установлена в глобальный Python)

## Установка

```bash
# Установка зависимостей (требуются права администратора)
pip install -r requirements.txt
```

## Запуск

```bash
cd python_arcade_sim
python main.py
```

## Проверка работы

При запуске открывается окно 1280x720 с заголовком "Ball impact simulator (Python + arcade)" и текстом:
> "Baseline window: arcade runs correctly."

Фон: тёмный серо-зелёный (DARK_SLATE_GRAY).
