# Шаг 8: Физика мяча — динамика, интегратор, энергетическая защита

## Цель

Реализовать динамику мяча, интегратор semi-implicit Euler, энергетическую защиту и пост-контактный полёт.

## Созданные модули

### `physics/ball.py`

#### Уравнения движения

**Линейные ускорения (второй закон Ньютона):**
```
a_x = F_t / m
a_y = F_n / m - g
```

**Угловое ускорение (момент силы):**
```
omega_dot = -(F_t * r) / I
```

где:
- `I = if * m * r²` — момент инерции
- `if = 0.4` для сплошного шара
- `if = 2/3` для полого шара
- Знак минус: положительная F_t создаёт вращение против часовой стрелки

#### Интегратор (semi-implicit Euler)

```python
# 1. Сначала скорости
v_x += a_x * dt
v_y += a_y * dt
omega += omega_dot * dt

# 2. Затем позиции
x += v_x * dt
y += v_y * dt
phi += omega * dt
```

**Преимущества semi-implicit Euler:**
- Лучше сохраняет энергию в колебательных системах
- Более стабилен для жёстких пружин
- Симплектический метод (сохраняет фазовый объём)

#### Энергетическая защита (clampRebound)

**Проблема:** При численном интегрировании может возникать искусственный прирост энергии:
- Мяч отскакивает с большей скоростью, чем до удара
- Вращение увеличивается без физической причины

**Решение:**
1. Вычисляем текущую кинетическую энергию:
   ```
   KE = 0.5 * m * (v_x² + v_y²) + 0.5 * I * omega²
   ```

2. Если KE > KE_initial * 0.999:
   - Масштабируем скорости: `v *= sqrt(KE_target / KE)`

3. Приоритетное ограничение (`clamp_rebound_priority`):
   - Сначала ограничиваем линейную энергию (90%)
   - Затем вращательную (10%)

#### Пост-контактный полёт

```python
# Гравитация
v_y += -g * dt

# Позиция
x += v_x * dt
y += v_y * dt

# Затухание вращения
omega *= (1 - k_spin * dt)
```

**Затухание вращения:**
- Физический смысл: сопротивление воздуха
- `k_spin = 0.1` 1/с (калибровка)

#### Клиппинг скоростей

Защита от численной нестабильности:
- `|v_x|, |v_y| <= K_VELOCITY_CAP` (100 м/с)
- `|omega| <= K_VELOCITY_CAP / r`

## Структуры данных

### BallForces

| Параметр | Описание |
|----------|----------|
| fn | Нормальная сила (положительная = вверх), Н |
| ft | Касательная сила (положительная = вправо), Н |

## Функции

### `compute_ball_accelerations(ball_state, ball_params, forces)`

Вычислить ускорения мяча.

**Returns:** `(a_x, a_y, omega_dot)`

### `integrate_ball(ball_state, ball_params, accelerations, dt)`

Интегрировать уравнения движения (semi-implicit Euler).

Обновляет `ball_state` на месте.

### `clamp_rebound(ball_state, ball_params, ke_initial)`

Ограничить кинетическую энергию (простое масштабирование).

**Returns:** `True`, если энергия была ограничена.

### `clamp_rebound_priority(ball_state, ball_params, ke_initial)`

Ограничить кинетическую энергию с приоритетом линейной скорости.

**Returns:** `True`, если энергия была ограничена.

### `step_ball_post_flight(ball_state, ball_params, dt)`

Шаг пост-контактного полёта (гравитация + затухание вращения).

### `compute_ball_kinetic_energy(ball_state, ball_params)`

Вычислить кинетическую энергию мяча.

**Returns:** KE, Дж.

## Ожидаемые эффекты

1. **При ударе:**
   - Fn ускоряет мяч вверх
   - Ft ускоряет мяч горизонтально и создаёт вращение
   - Гравитация постоянно действует вниз

2. **Вращение:**
   - Положительная Ft (вправо) создаёт отрицательное omega_dot (по часовой)
   - Отрицательная Ft (влево) создаёт положительное omega_dot (против часовой)

3. **Энергетическая защита:**
   - При искусственном разгоне скорости масштабируются вниз
   - Сохраняется соотношение между линейной и вращательной энергией

4. **Пост-контактный полёт:**
   - Мяч летит по параболе под действием гравитации
   - Вращение постепенно затухает
   - Горизонтальная скорость постоянна (без сопротивления воздуха)
