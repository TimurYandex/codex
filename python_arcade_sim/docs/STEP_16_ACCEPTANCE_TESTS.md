# Шаг 16: Проверка критериев приемки

## Критерии приемки

### 1. Угол входа влияет на траекторию и угол выхода

**Проверка:**
- Запустить симуляцию с углом -30°
- Запустить симуляцию с углом -60°
- Сравнить траектории и углы выхода

**Ожидаемый результат:**
- Больший угол входа → более крутая траектория
- Угол выхода отличается от угла входа (из-за трения)

### 2. Вращение визуально видно и соответствует знаку ω

**Проверка:**
- Запустить с spin = 100 rad/s, dir = cw
- Запустить с spin = 100 rad/s, dir = ccw
- Наблюдать метку на мяче и стрелку направления

**Ожидаемый результат:**
- Метка вращается по часовой (cw) или против (ccw)
- Стрелка направления показывает правильное направление

### 3. Ракетка по X центрирована, удар по центру кадра

**Проверка:**
- Запустить симуляцию
- Наблюдать позицию удара

**Ожидаемый результат:**
- Мяч пересекает поверхность в центре окна по X

### 4. Анимация читаема и замедлена (default 0.005)

**Проверка:**
- Запустить симуляцию с time_scale = 0.005
- Наблюдать скорость анимации

**Ожидаемый результат:**
- Симуляция длится несколько секунд
- Детали контакта различимы
